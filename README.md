# Number Gate Game

A browser-based strategy game inspired by Last War. Guide your squad through red and blue number gates on a suspension bridge, then battle waves of enemies.

## How to Play

- **Drag left/right** to move your squad on the bridge
- **Red gates** reduce your unit count (e.g. -3)
- **Blue gates** multiply your unit count (e.g. x2)
- Your soldiers automatically shoot at approaching enemies
- If your units reach **zero**, you lose
- Clear all waves to advance to the next level

## Tech Stack

Pure HTML + CSS + Canvas (no dependencies). Runs in any modern browser.

## Play Online

Open `index.html` in a browser, or visit the GitHub Pages deployment.
